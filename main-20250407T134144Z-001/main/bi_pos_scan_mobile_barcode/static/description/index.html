POS Mobile Barcode and QR Code Scanner Odoo Apps

POS Barcode and QR Code Scanner with Mobile Odoo App helps users to scan QR code/barcode using mobile. Users can configure QR code/barcode in that scan POS product by internal reference, barcode, QR code or all. Users can scan products using the camera option or select an image file. Users can enable the 'continue scan product' checkbox so the product will be scanning continuously in POS and also get the notification when product successfully or fails to scanned.

Features

POS Configuration for QR Code or Barcode

User can configure or use product scan option in mobile.

Easy to Scan QR Code/Barcode with Mobile

User can easily scan QR code/barcode with using mobile.

Continuously Scanning Product

User can enable 'continue scan product' checkbox so product will be scanning continuously in POS.

Popup Notification when Product Scan

User will get the notification when product successfully scanned or failed to scanned.

POS Configuration for Barcode/QR Scanner

You can scan POS product by internal reference, barcode, QR code and use option all.

You can use or select checkboxes of product success notification, product failed notification, product success play sound, product failed play sound and also use continue scan product option.

Click on 'Start' Button

In POS, when you click on 'start' button, scan QR code wizard will be open.

Scan QR Code Wizard

You can use scan an image file option in that select QR code or barcode image file.

You can select or browse image file.

Product Successfully Scanned by QR Code

You can see product successfully scanned by QR code with success notification and also automatically added to the cart.

You can click on 'Request Camera Permissions' option for open camera to scan QR code or barcode.

Product Successfully Scanned by QR Code

You can see product successfully scanned by QR code with success notification and also automatically added to the cart.

Product Failed to Scanned by Barcode

You can see warning message when product failed to scanned by barcode/internal reference or not exist in product.
