Sybyl ESD POS & Account Novitus Printer

Sybyl ESD POS & Account for Novitus Printer

Account's Configuration:
  • Set IP and PORT, in accounting Configuration under ESD Printer Configuration.
  • Set "PTU Value" in Account tax.
  • Tax must be "Included in Price" in Kenya.
  • In Product Set "HS Code" value and set Factor Type based on tax.
  • Under Accounting -> Configuration -> Settings -> Taxes, need to set PTU Value and Factor type.
  • Under Accounting -> Configuration -> Payment Methods, Need to set 'Fiscal Printer Payment Type' Value.
POS's Configuration:
  • You can send individual IP for individual Shops. In POS Shop, Configuration under Direct Devices. Enable "Direct Devices" and added IP and PORT.
  • Set "PTU Value" in Account tax.
  • Tax must be "Included in Price" in Kenya.
  • In Product Set "HS Code" value and set Factor Type based on tax.
  • Under Accounting -> Configuration -> Settings -> Taxes, need to set PTU Value and Factor type.
  • Under POS -> Configuration -> Payment Methods, Need to set 'Fiscal Printer Payment Type' Value.
